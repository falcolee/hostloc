# 应用名
开源android hostloc论坛APP，造福广大MJJ

已经实现的功能有：
- [x] 首页热门帖子
- [x] 分区浏览
- [x] 登录后帖子显示签名
- [x] 回复帖子
- [x] 个人中心，积分、等级、威望数据
- [x] 我的私信
- [x] 收藏
- [x]
