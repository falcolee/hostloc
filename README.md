# 应用名
开源android hostloc论坛APP，造福广大MJJ
> 感谢fabricezhang大大的LCG项目，该项目在此开源项目的基础上改造而成
  [lcg](https://github.com/fabricezhang/lcg)

已经实现的功能有：

- [x] 首页热门帖子
- [x] 分区浏览
- [x] 帖子详情、登录后帖子显示签名
- [x] 回复帖子
- [x] 个人中心，积分、等级、威望
- [x] 我的私信
- [x] 我的收藏
- [x] 浏览记录
- [x] 个人资料查看、头像放大下载
- [x] 更多
